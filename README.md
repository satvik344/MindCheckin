# MindCheck

AI-powered stress analysis web application built using TensorFlow.js.

## Features

- Stress prediction using an Artificial Neural Network
- TensorFlow.js client-side inference
- Interactive questionnaire
- Responsive user interface
- Dark mode support

## Technologies

- HTML
- JavaScript
- TensorFlow.js
- Tailwind CSS
